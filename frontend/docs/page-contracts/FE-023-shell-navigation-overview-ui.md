## 本次实现

- 新增 `system.overview` 路由：`/systems/:systemId/overview`。
- 进入系统后默认跳转系统总览，不再直接进入系统资料配置页。
- 侧边栏按平台层和系统层分层展示：系统内页面隐藏平台系统/账号/角色配置入口，平台层保留平台入口。
- 侧边栏增加当前系统卡片和“切换系统”入口。
- 系统总览展示当前系统上下文、配置进度、推荐路径和快捷入口。

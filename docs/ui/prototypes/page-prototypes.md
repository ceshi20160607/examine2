# P12 页面级原型说明

状态：frozen-for-FE-024

本文件是 `docs/ui/ui-design.md` 的页面级落地稿。FE-024 必须按本文件改造页面，不再由前端临场决定页面结构。

## 1. 平台层

### 1.1 我的系统

```text
┌──────────────────────────────────────────────────────────────┐
│ 我的系统                                      [创建系统]       │
│ 选择要进入的系统，或创建新的自定义系统。                     │
├──────────────────────────────────────────────────────────────┤
│ 搜索系统名称/编码 [__________] [查询]                         │
├──────────────────────┬──────────────────────┬────────────────┤
│ 系统卡片             │ 系统卡片             │ 空态/更多       │
│ 名称 / 编码          │ 名称 / 编码          │                │
│ 状态 / 租户模式      │ 状态 / 租户模式      │                │
│ [进入系统]           │ [进入系统]           │                │
└──────────────────────┴──────────────────────┴────────────────┘
```

规则：

- 主按钮是“创建系统”。
- 系统卡片主操作是“进入系统”，不是整行隐式点击。
- 空态文案：`暂无可用系统。创建系统后，你会成为该系统的超级管理员。`

### 1.2 平台系统/账号/角色/配置

```text
┌──────────────────────────────────────────────────────────────┐
│ 页面标题                                      [主操作]         │
│ 页面说明 / 当前权限提示                                      │
├──────────────────────────────────────────────────────────────┤
│ 筛选：关键字 [____] 状态 [全部 v]              [查询] [重置]  │
├──────────────────────────────────────────────────────────────┤
│ 表格：对象名称 | 编码/账号 | 状态 | 更新时间 | 操作          │
└──────────────────────────────────────────────────────────────┘
┌─────────────────────────────── 右侧抽屉 ─────────────────────┐
│ 新建/编辑表单，保存/取消固定在底部                            │
└──────────────────────────────────────────────────────────────┘
```

规则：

- 新建和编辑统一使用右侧抽屉。
- 启停、重置密码、授权必须二次确认或在抽屉中完成。
- 平台配置敏感值默认脱敏。

## 2. 系统总览

```text
┌──────────────────────────────────────────────────────────────┐
│ 系统总览：{系统名称}                         [切换系统]       │
│ 当前租户 / 当前成员 / 权限数量 / 系统状态                     │
├──────────┬──────────┬──────────┬──────────┬──────────┬───────┤
│ 成员数   │ 部门数   │ 角色数   │ 应用数   │ 模块数   │ 已发布 │
├──────────────────────────────────────────────────────────────┤
│ 推荐路径：1 系统设置 -> 2 应用建模 -> 3 运行填报 -> 4 协同集成 │
├──────────────────────┬──────────────────────┬────────────────┤
│ 快捷入口：成员权限   │ 快捷入口：应用配置   │ 快捷入口：运行台 │
└──────────────────────┴──────────────────────┴────────────────┘
```

规则：

- 进入系统后的默认页必须是系统总览。
- 总览只负责引导和汇总，不承载复杂编辑表单。
- 没有租户、成员或权限时，总览显示阻断提示和修复入口。

## 3. 系统设置

### 3.1 成员

```text
┌──────────────────────────────────────────────────────────────┐
│ 成员                                      [邀请成员]          │
├──────────────────────────────────────────────────────────────┤
│ 筛选：登录名/姓名 [____] 部门 [全部] 角色 [全部] [查询]       │
├──────────────────────────────────────────────────────────────┤
│ 成员 | 登录名 | 部门 | 角色 | 状态 | 操作                     │
└──────────────────────────────────────────────────────────────┘
右侧抽屉：邀请/编辑成员
  平台账号登录名
  显示名
  租户
  部门
  岗位
  角色
```

规则：

- 邀请成员必须先选择或输入平台账号。
- 分配角色使用多选，不使用 prompt。
- 停用成员必须填写原因或显示默认原因。

### 3.2 部门

```text
┌────────────── 部门树 ──────────────┬──────────── 部门详情 ────┐
│ [新增部门]                         │ 名称 / 编码 / 排序       │
│ 总部                               │ 上级部门                 │
│  ├─ 业务部                         │ 关联成员数               │
│  └─ 财务部                         │ [编辑] [删除]            │
└────────────────────────────────────┴─────────────────────────┘
```

规则：

- 有子部门或成员时删除按钮禁用并说明原因。
- 新增/编辑使用右侧抽屉。

### 3.3 角色授权

```text
角色列表：角色名 | 状态 | 成员数 | 操作
授权抽屉：
  Tabs: 菜单权限 | 操作权限 | 数据范围 | 字段权限
  底部：[保存授权] [取消]
```

规则：

- 权限按模块分组，不展示一整串逗号文本。
- 保存后刷新权限版本。

### 3.4 字典

```text
┌──── 字典类型 ────┬──────────── 字典项 ────────────┐
│ 类型列表         │ 当前类型：xxx                   │
│ [新增类型]       │ [新增字典项]                    │
│                  │ 字典项 | 值 | 状态 | 排序 | 操作 │
└──────────────────┴────────────────────────────────┘
```

规则：

- 左侧选择类型，右侧维护字典项。
- 删除前必须先检查引用，引用存在时禁用删除。

## 4. 应用配置

```text
┌──────────────────────────────────────────────────────────────┐
│ 应用配置：{应用} / {模块}                                    │
├────────── 步骤条 ────────────────────────────────────────────┤
│ 1 应用 -> 2 模块 -> 3 字段 -> 4 页面 -> 5 发布检查 -> 6 发布  │
├──────────────────────────────┬───────────────────────────────┤
│ 当前步骤主内容               │ 发布检查 / 选择摘要 / 帮助提示 │
└──────────────────────────────┴───────────────────────────────┘
```

规则：

- 没有应用时只展示应用创建引导。
- 没有模块时字段、页面、发布入口禁用。
- 发布检查失败项要能定位到字段或页面配置。

### 4.1 字段配置

表格字段：

- 字段名称、字段编码、类型、必填、唯一、列表显示、表单可写、状态、操作。

右侧抽屉：

- 基础信息：名称、编码、类型。
- 校验：必填、唯一、长度/数值范围。
- 显示：列表、表单、详情。
- 选项：字典/枚举选项。

### 4.2 页面配置

Tabs：

- 列表视图：字段列、排序、筛选字段、行操作。
- 表单视图：字段分组、必填提示、默认值。
- 详情视图：字段分组、附件、审批、历史。
- 菜单动作：运行台菜单、按钮权限。

## 5. 运行台

### 5.1 运行列表

```text
左侧：已发布模块菜单
主区：
  标题：{模块名称}                         [新建记录] [导出]
  筛选区：基础筛选 + 高级筛选折叠
  表格：标题字段 | 状态 | 创建人 | 更新时间 | 关键字段 | 操作
```

规则：

- 运行台只显示已发布模块。
- 没有已发布模块时显示“请先在应用配置中发布模块”。
- 配置按钮不放在运行台主操作里。

### 5.2 记录详情

```text
┌──────────────────────────────────────────────────────────────┐
│ {记录标题} 状态标签                         [编辑] [提交审批] │
├──────────────────────────────────────────────────────────────┤
│ Tabs: 详情 | 附件 | 审批 | 历史 | 关联                      │
└──────────────────────────────────────────────────────────────┘
```

规则：

- 流程锁定时编辑和删除禁用。
- 提交审批必须二次确认。

## 6. 流程

### 6.1 流程模板

```text
模板列表 | 状态 | 绑定模块 | 当前版本 | 操作
右侧：模板详情 + 默认流程图 + 发布检查
```

规则：

- 发布前必须展示检查结果。
- 绑定模块必须选择已发布或可绑定模块。

### 6.2 流程工作台

Tabs：

- 待办、抄送、我的申请、流程实例。

待办表格：

- 标题、来源模块、发起人、到达时间、状态、操作。

审批抽屉：

- 业务摘要、审批历史、意见、动作按钮。

## 7. 文件与导出

### 7.1 文件中心

布局：

- 顶部上传区。
- 下方文件列表。
- 详情抽屉展示引用对象、预览、下载、删除原因。

### 7.2 导出任务

Tabs：

- 导出模板、导出任务。

规则：

- 创建任务前必须显示当前模块发布状态。
- 任务失败要显示失败原因和重试按钮。

## 8. OpenAPI

页面：

- 客户端列表。
- 详情抽屉 Tabs：基础信息、凭证、Scope、IP 白名单、限流、调用日志。

规则：

- Secret 只展示一次。
- Scope 按业务域分组选择。
- 调用日志支持 requestId 检索。

## 9. 审计运维

审计页：

- 顶部 requestId 搜索。
- 日志类型 Tabs。
- 详情抽屉展示请求上下文和错误码。

运维页：

- 健康卡片。
- 配置检查。
- 版本信息。
- migration 状态。

异常项置顶，给出处理建议。

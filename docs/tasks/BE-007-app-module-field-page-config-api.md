# BE-007 应用模块字段页面配置 API

- taskId: BE-007
- 标题: 应用模块字段页面配置 API
- 负责角色: backend
- 所属大任务/模块: 后端 / 应用配置中心
- 目标: 实现 APP、MOD、FIELD、UI 接口组和发布检查。
- 输入文件: `docs/api.md`、`docs/db_design.md`、`backend/examine-module/`
- 输出文件或输出目录: `backend/examine-module/`

## 详细工作内容

- 实现应用、模块、字段、字段类型、列表视图、表单、详情、菜单、动作配置接口。
- 实现模块发布检查和发布版本生成。
- 校验字段编码、字段类型、关联配置、页面字段存在性和权限配置。

## 完成状态定义

- 默认状态: pending。
- 完成条件: 应用配置中心 API 可生成运行态发布版本。

## 验收标准

- 草稿和发布版本语义清晰，运行态只读取发布版本。
- 发布检查失败返回可定位配置项和 requestId。

## 测试/自检要求

- 测试字段编码重复、字段类型非法、页面引用不存在字段、发布版本冲突。

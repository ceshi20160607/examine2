# BE-003 认证会话安全

- taskId: BE-003
- 标题: 认证会话安全
- 负责角色: backend
- 所属大任务/模块: 后端 / 认证会话
- 目标: 实现 AUTH-001 至 AUTH-005 MVP 认证会话接口。
- 输入文件: `docs/api.md`、`docs/db_design.md`、`backend/examine-core/`
- 输出文件或输出目录: `backend/examine-plat/`、`backend/examine-web/`

## 详细工作内容

- 实现注册、登录、刷新、退出、当前用户接口。
- 校验账号正常、停用、锁定状态和登录失败限制。
- 写入登录/退出日志并返回统一响应。

## 完成状态定义

- 默认状态: pending。
- 完成条件: AUTH MVP 接口符合冻结 API。

## 验收标准

- 平台用户是唯一登录主体。
- 认证失败错误码、requestId 和前端提示可断言。

## 测试/自检要求

- 单元测试密码错误、账号停用/锁定、token 过期、退出幂等。

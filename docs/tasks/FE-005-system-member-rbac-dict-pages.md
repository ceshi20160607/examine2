# FE-005 系统成员 RBAC 字典页面

- taskId: FE-005
- 标题: 系统成员 RBAC 字典页面
- 负责角色: frontend
- 所属大任务/模块: 前端 / 系统管理
- 目标: 实现系统资料、租户、成员、部门、角色权限和字典管理页面。
- 输入文件: `docs/api.md`、`frontend/src/api/`
- 输出文件或输出目录: `frontend/src/pages/system/`、`frontend/docs/page-contracts/FE-005-system-member-rbac-dict-pages.md`

## 详细工作内容

- 实现 SYS、MEM、RBAC、DICT 接口对应页面。
- 支持部门树、成员列表、成员角色、角色授权、权限目录、数据范围和字段权限配置。
- 支持字典类型、字典项、启停、引用情况和删除限制展示。
- 补齐页面级 API 映射证据，覆盖路由、API ID、必填入参、响应字段、上下文依赖、枚举/状态/错误码、权限禁用态、空态/错误态、requestId 展示和无旁路请求检查。

## 完成状态定义

- 默认状态: pending。
- 完成条件: 系统管理员能维护组织、角色、权限和字典，`FE-005` 页面级契约证据文件已补齐。

## 验收标准

- 平台账号字段只读，系统成员扩展字段可按权限编辑。
- 无授权权限时授权控件禁用并说明原因。
- 权限禁用态、空态、错误态和完成证据可被 FE-012 汇总到 `frontend/docs/api-contract-map.md`。

## 测试/自检要求

- 自检跨租户禁用、角色停用、字典引用限制、权限缓存刷新提示。

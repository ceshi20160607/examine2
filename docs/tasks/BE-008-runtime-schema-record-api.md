# BE-008 运行 schema 与记录 API

- taskId: BE-008
- 标题: 运行 schema 与记录 API
- 负责角色: backend
- 所属大任务/模块: 后端 / 应用运行台
- 目标: 实现 RUN-001 至 RUN-010 运行台菜单、schema、记录查询、保存、详情、历史和关联接口。
- 输入文件: `docs/api.md`、`docs/db_design.md`、`backend/examine-module/`、`backend/examine-core/`
- 输出文件或输出目录: `backend/examine-module/`

## 详细工作内容

- 读取发布版本生成运行 schema 和菜单。
- 实现记录新增、编辑、软删除、详情、历史、关联和字段值保存。
- 处理字段权限、动态字段校验、唯一性、附件引用、流程锁定和运行记录写入幂等。

## 完成状态定义

- 默认状态: pending。
- 完成条件: 运行台核心记录接口符合 API 契约。

## 验收标准

- 字段无写权限直接拒绝，不静默忽略。
- 业务记录保存事务覆盖主记录、字段值、索引、历史、关联和附件引用。

## 测试/自检要求

- 测试必填缺失、类型错误、唯一冲突、数据范围越权、流程锁定、软删除。

# DBA-003 模块配置运行记录导出表设计

- taskId: DBA-003
- 标题: 模块配置运行记录导出表设计
- 负责角色: dba
- 所属大任务/模块: DB 设计 / 动态模块
- 目标: 设计应用、模块、字段、页面、运行记录、EAV、历史和导出任务表。
- 输入文件: `docs/prd.md`、`docs/api.md`、`docs/service_info.md`、`docs/legacy_reference.md`

## 详细工作内容

- 设计应用、应用版本、模块、字段、页面、菜单、动作和发布版本表。
- 设计运行记录、字段值、索引值、关联关系、子表行和历史快照表。
- 设计导出模板、导出任务、筛选快照、权限快照和结果文件引用。

## 完成状态定义

- 默认状态: pending。
- 完成条件: `docs/db_design_parts/module-runtime-export.md` 中 `un_module_` 表设计覆盖配置态、发布态、运行态和导出闭环；最终 `docs/db_design.md` 只由 DBA-005 汇总写入。

## 验收标准

- 运行态只读取发布版本。
- 动态字段值不暴露物理 EAV 结构给前端 API。

- 自检唯一字段、组合唯一、软删除复用和历史快照规则。
- 自检导出任务状态流转和结果文件引用。

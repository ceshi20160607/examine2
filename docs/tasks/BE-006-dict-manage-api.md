# BE-006 字典管理 API

- taskId: BE-006
- 标题: 字典管理 API
- 负责角色: backend
- 所属大任务/模块: 后端 / 系统字典
- 目标: 实现 DICT-001 至 DICT-011 系统字典类型和字典项管理接口。
- 输入文件: `docs/api.md`、`docs/db_design.md`、`backend/examine-module/`
- 输出文件或输出目录: `backend/examine-module/`

## 详细工作内容

- 实现字典类型列表、创建、更新、启停、删除和使用情况查询。
- 实现字典项列表、创建、更新、启停、删除。
- 维护内置只读、引用限制、层级关系和缓存版本刷新。

## 完成状态定义

- 默认状态: pending。
- 完成条件: 字典接口能支撑字段配置和运行表单可选项。

## 验收标准

- 停用字典项保留历史展示快照。
- 被字段或记录引用时按契约限制删除。

## 测试/自检要求

- 测试编码唯一、内置只读、引用限制、缓存刷新、树形字典。

## 依赖任务

- BE-005

## 可并行关系

- 不可与依赖字典能力的 BE-007 同批执行；完成后启动 BE-007。

## 不允许事项

- 不直接物理删除已引用字典数据。
- 不把字典配置混入平台字段类型元数据定义。

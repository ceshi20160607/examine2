# FE-008 应用运行台页面

- taskId: FE-008
- 标题: 应用运行台页面
- 负责角色: frontend
- 所属大任务/模块: 前端 / 应用运行台
- 目标: 实现运行菜单、业务列表、业务表单、详情、历史、关联和提交审批入口。
- 输入文件: `docs/api.md`、`frontend/src/api/`
- 输出文件或输出目录: `frontend/src/pages/runtime/`、`frontend/docs/page-contracts/FE-008-runtime-workbench-pages.md`

## 详细工作内容

- 实现 RUN-001 至 RUN-010 页面闭环。
- 支持分页、筛选、排序、新增、编辑、软删除、提交审批、历史、关联数据。
- 处理无权限、无可见字段、流程锁定、版本变化、错误 requestId 展示。
- 补齐页面级 API 映射证据，覆盖路由、API ID、必填入参、响应字段、上下文依赖、枚举/状态/错误码、权限禁用态、空态/错误态、requestId 展示和无旁路请求检查。

## 完成状态定义

- 默认状态: pending。
- 完成条件: 业务用户可按权限完成运行台核心填报，`FE-008` 页面级契约证据文件已补齐。

## 验收标准

- 保存/审批/删除后刷新当前页并保留筛选。
- 字段权限和按钮权限按后端返回状态展示。
- 权限禁用态、空态、错误态和完成证据可被 FE-012 汇总到 `frontend/docs/api-contract-map.md`。

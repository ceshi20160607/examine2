# BE-011 导出任务 API

- taskId: BE-011
- 标题: 导出任务 API
- 负责角色: backend
- 所属大任务/模块: 后端 / 导出中心
- 目标: 实现导出模板、导出任务、任务查询、重试、取消和结果文件权限。
- 输入文件: `docs/api.md`、`docs/db_design.md`、`backend/examine-module/`、`backend/examine-upload/`
- 输出文件或输出目录: `backend/examine-module/`

## 详细工作内容

- 实现 EXP-001 至 EXP-008。
- 创建导出任务时保存筛选快照、权限快照、脱敏规则和任务状态。
- 后台生成失败时更新失败原因，成功后绑定结果文件。

## 完成状态定义

- 默认状态: pending。
- 完成条件: MVP 导出任务闭环可追踪、可下载、可重试。

## 验收标准

- 无导出权限或字段无导出权限时拒绝或脱敏。
- 处理中任务不重复领取，失败任务按规则重试。

## 测试/自检要求

- 测试导出权限、任务状态流转、失败重试、取消、结果文件下载权限。

# BE-010 上传文件引用 API

- taskId: BE-010
- 标题: 上传文件引用 API
- 负责角色: backend
- 所属大任务/模块: 后端 / examine-upload
- 目标: 实现文件上传、列表、详情、预览、下载、删除和业务引用权限。
- 输入文件: `docs/api.md`、`docs/db_design.md`、`backend/examine-upload/`
- 输出文件或输出目录: `backend/examine-upload/`

## 详细工作内容

- 实现 FILE-001 至 FILE-006。
- 保存文件元数据、临时/已引用/已删除/过期状态和引用关系。
- 下载、预览必须校验业务对象权限和文件引用关系。

## 完成状态定义

- 默认状态: pending。
- 完成条件: 文件接口可被运行台附件和导出结果文件复用。

## 验收标准

- 临时文件绑定失败按规则清理或过期处理。
- 已引用文件不允许直接物理删除。

## 测试/自检要求

- 测试类型大小限制、存储不可用、无业务权限下载、引用文件删除限制。

## 可并行关系

- 可与 BE-009 并行；BE-011 与 BE-012 必须等待本任务完成。

## 不允许事项

- 不把对象存储失败静默吞掉。

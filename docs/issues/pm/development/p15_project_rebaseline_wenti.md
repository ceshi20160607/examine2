阶段：development

状态：open

## PM 总裁决

P14 `fullProjectDeployable=true` 结论撤回。当前系统不是功能小缺口，而是用户可用性、信息架构、登录落点、成员开通、页面主动作和验收方式整体不达标。

P15 在冻结 `docs/product/p15-project-rebaseline.md`、UI/UX 设计和新验收剧本前，不允许继续生成最终部署包。

## P15 不允许事项

1. 不允许继续把 P14 包作为完整系统交付。
2. 不允许先局部修补页面再回头补 PM 设计。
3. 不允许只用平台管理员验收。
4. 不允许只用接口或局部 E2E 验收。
5. 不允许继续把系统成员开通流程暴露成平台账号和系统成员两段操作。
6. 不允许没有退出、分页、新建主入口就打包。

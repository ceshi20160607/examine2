## 发现与修正

- 第一次发布失败：`MODULE_CONFIG_VERSION_CONFLICT`。
- 原因：保存字段、页面、菜单和动作后，模块版本已经从创建时的 `1` 增长到 `9`，发布必须重新读取模块详情后使用最新版本。
- 第二次执行已修正为发布前调用模块详情接口，发布通过。

## 保留风险

- 本记录是 API 级 E2E，不等价于浏览器点击流 E2E。
- CDP 浏览器登录流在当前 Windows/Chrome headless 环境未稳定进入认证态，不能据此判定浏览器 E2E 通过。
- 因此 P14 打包闸门继续保持 blocked，后续必须补浏览器点击流或可替代的稳定 UI E2E。

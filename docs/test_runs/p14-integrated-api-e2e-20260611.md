## 发现与修正

- 第一次发布失败：`MODULE_CONFIG_VERSION_CONFLICT`。
- 原因：保存字段、页面、菜单和动作后，模块版本已经从创建时的 `1` 增长到 `9`，发布必须重新读取模块详情后使用最新版本。
- 第二次执行已修正为发布前调用模块详情接口，发布通过。
